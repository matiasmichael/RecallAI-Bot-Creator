==================================================
RecallAI Bot Creator - Setup Instructions
==================================================

This script creates a RecallAI bot that joins your Zoom or Teams meeting.

REQUIREMENTS
------------
- Python 3.10 or higher
- Internet connection

SETUP (One-time)
----------------
1. Open Terminal (Mac) or Command Prompt (Windows)

2. Install the required dependency:

   pip install httpx

   If you get a permission error, try:

   pip install --user httpx

RUNNING THE SCRIPT
------------------
1. Open Terminal (Mac) or Command Prompt (Windows)

2. Navigate to the folder where you saved the script. For example:

   cd ~/Documents/RecallAI-Bot-Creator

3. Run the script:

   python3 create_bot.py

4. When prompted, paste your Zoom or Teams meeting link and press Enter

5. The bot will join your meeting!

TROUBLESHOOTING
---------------
- "command not found: python3"
  Try using "python" instead of "python3"

- "No module named httpx"
  Run: pip install httpx

- "Python version too old"
  Download Python 3.10+ from https://www.python.org/downloads/

SUPPORT

Contact your administrator if you encounter issues.
